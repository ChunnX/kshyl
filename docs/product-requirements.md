# 产品需求草案

## 核心目标

让家庭成员用最少操作记录声音、故事、孩子成长和家庭日常，系统自动保存、转写、整理，家人负责校对，最终生成可打印的回忆录或成长册。

## MVP 功能

- 全龄友好的一键录音
- 语音陪聊：对话模式 / 倾诉模式
- AI 根据前文和历史故事自然追问
- 照片上传，作为故事线索或成长记录素材
- 用户关系：定义家庭成员及其关系
- 主题记录：每个主题可独立完成或邀请家人共创
- 邀请补充：可邀请某人对主题或某段故事补充细节
- 录音上传与转写
- AI 整理故事草稿
- 家人校对故事
- 生成书稿摘要

## 二期功能

- 图文混排成书
- 时间线与人物关系
- 订阅消息提醒
- 数字人格对话
- 声音克隆与 TTS
