# 隐私与授权要求

- 所有音频、文本和书稿默认私密。
- 声音克隆必须取得本人明确授权，建议保存授权音频。
- 纪念对话必须明确提示“这是 AI 生成，不是真人本人”。
- 数字人格只能基于已授权、已校对内容回答。
- 用户必须能删除录音、故事、声音模型和导出文件。
- 对外分享应使用可撤销的授权链接。

## 落地状态

已实现：

- 登录 + JWT 鉴权，人物资源归属校验（防越权）。
- 纪念对话 403 同意闸门 + 仅 `approved` 故事 + AI 免责提示（响应 `disclaimer`）。
- 删除链路：录音 / 故事 / 照片 / 书稿，以及「删除整个人物及其全部数据」级联接口。
- 可撤销分享链接：撤销后读取与补充返回 410。

待办：

- 声音克隆授权音频留存与启用流程（当前 `voice-clone.service.js` 默认关闭）。
- 删除时同步清理对象存储中的物理文件（接入 COS 后补齐）。
- 实时语音 WebSocket 目前未校验 token（开发期靠 `DEV_AUTH_BYPASS`），上线前需在握手时验证 JWT。
